Spiritual Names & terms List compiled by Sāmudroprem सामुद्रप्रेम् Email: AwakenIMAP@gmail.com
 N.B. Pronunciation: Ā ā = long ‘a’ like 'ah' but not quite ‘ar’ (spelling alternative 'aa') / ê = ai / é = ay / Ī, ī = spelling alternative 'ee' / Ū ū = spelling alternative 'oo'
e.g. Ānanda can be spelled as Aananda. N.B. Anand = Joyless. Ānanda (Aananda) = Joy, Bliss
Not all Sanskrit, or other languages, pronounciations are always defined here e.g. no difference is given between sh and zh. The Anusvara letter n is given as m and n, e.g. Samsara and Sansara. For correct pronounciation see Sanskrit dictionaries.
There is no 'f' sound in Sanskrit, and 'ph' is prononced as 'p-h' as in 'uphill'.
Source abbreviations: OD = Osho darshan diaries / OB = Osho discourse books / MW = Monier-Williams Sanskrit Dictionary / S = previous Osho Sannyas name lists / SD = other Sanskrit dictionaries / HD = Hindi dictionaries PD = Pali dictionaries
TS = Thousand Name of Shiva / TB = Thousand Names of Brahma / TV = Thousand Names of Vishnu / Onl = Online resources / 99N = 99 Names of Allah / Sp = Samudroprem's addition or confirmation
Note 1: "Add Veet or Mukta" = Recommendation to combine with 1st Name "Veet" to mean "Beyond" or "Transcendence of"; or add 1st Name "Mukta" to mean "Freedom from."
Note 2: Neutral names can end in an 'o', however in traditional Sanskrit the 'o' joins two words together as a compund name e.g. Sāmudroprem
Note 3: In Sanskrit most male names end with an 'a', however in Hindi the same name can be pronounced (and written in script) without the 'a' and end in the consonant. Thus for male names the final 'a' is optional.
Note 4: The difference of Sanskrit and Hindi: Sanksrit 'v' is oftern pronounced in Hindi as a 'w,' e.g. Svabhava or Swabhav. Also sometimes a "V' is also pronounced as a 'B' e.g. Vanshi or Banshi.
Note 5: In Sanskrit most (but not all) female names with end with the long ā (aa). The long 'ī' (ee) is also a common feminine ending.
